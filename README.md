# Web Development Portfolio 🚀

Welcome to my master web development repository! This repository acts as a central hub (Monorepo/Master Directory) that organizes and links all of my individual web development projects, including Full-Stack and MERN applications.

## 📌 About This Repository
Instead of cluttering the profile, this repository uses **Git Submodules** to structurally connect separate projects into one organized space while fully preserving their original commit histories, contributions, and development timelines.

## 📂 Project Directory
Inside this repository, you will find dedicated folders for various applications, including:
* **Chat Applications** (Frontend & Backend separate submodules)
* **E-commerce Platforms** (Frontend & Backend architectures)
* **Full-Stack MERN Applications**
* **Frontend UI/UX Templates & API Integrations**

---
*Feel free to click on any folder to explore the individual repository, view the complete source code, and check out the commit history for that specific project!*
